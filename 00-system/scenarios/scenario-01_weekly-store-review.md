## 미션 1: 매장별 주간 매출 리뷰 + 본부장 압박 대비

장승웅(Big Blue) COO님의 요청. 본부장님이 표 보자마자 "이 매장 왜?" 물어볼 것.

> **장승웅(Big Blue)님**: 본부장님이 먼저 물어보실 거예요
> "지난 주 매장 어땠어? 어버이날 끝났으니까 반동도 봐야 돼"
>
> 단순히 "매출 X억" 식 답은 안 돼요
> "왜 그렇게 됐는지" 한 줄씩 있어야 해요. 매장별로

**(속으로)**: 본부장님 미팅에서 표 보여드리면 한 매장씩 손가락으로 짚어가며 물어보실 거다. 그 답을 표 옆에 미리 적어둬야 한다.

**분석 요청**:
```
@store_sales.csv @products_master.csv @voc_inflow.csv @campaign_calendar.csv
@templates/lush/weekly-store-review.md 관점으로
지난 주(5/12-5/18) 10개 매장 운영 리뷰 만들어줘. 본부장님이 한 매장씩 물어볼 거니까
매장별 한 줄 코멘트 + 본부장이 물어볼 질문 3개에 대한 답까지 미리 준비해줘.
```

**결과 확인 후 추가 질문 예시**:
```
"본부장님은 항상 '그래서 다음 주는 어떻게 할 거야?' 물어보세요.
표 보고 다음 주 액션 후보 2개만 뽑아주세요. GO/NO-GO 결정형으로."
```

## 미션 2: 매장별 VOC 칭찬 인용 추가

미션 1 끝내고 10시 15분. 정문혁(Geo)님의 VOC 데이터를 표 옆에 한 줄씩 박아넣으면 본부장 압박이 한층 약해진다.

> **정문혁(Geo)님**: 현숙님, voc_inflow.csv에서 매장별 칭찬·불만 비율 뽑아서 매장 코멘트 옆에 한 줄씩 넣으면 좋을 것 같아요
>
> "명동본점 — 매출 +18%, 칭찬 5건/불만 0건" 이런 식으로
> 본부장님이 매장 평가하실 때 매출 외에 응대 품질도 같이 보시거든요

**(속으로)**: 매출 숫자 옆에 사람 이야기 한 줄이 같이 들어가면 본부장님 톤이 부드러워진다. 작년에 그렇게 한 적이 있었다.

**분석 요청**:
```
@voc_inflow.csv
지난 주 매장 10곳에 대해 칭찬/불만 건수 표로 정리하고,
가장 칭찬 많이 받은 매장 1곳·불만 많이 받은 매장 1곳에 대해
실제 VOC 요약(summary 컬럼) 2~3건씩 직접 인용해줘.
```

**결과 확인 후 추가 질문 예시**:
```
"미션 1 매장 순위표 옆에 미션 2 VOC 칭찬·불만 건수 컬럼을 합쳐서
하나의 통합 표로 다시 만들어줘. 매장별로 매출 + 응대 품질이 한 번에 보이게."
```
