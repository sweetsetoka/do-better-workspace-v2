## 미션 1: 분기 1페이지 요약 (대표 결정용)

장승웅(Big Blue) COO님이 강조한 두 가지: "수수료 다 떼고 진짜 얼마" + "다음 분기 GO/NO-GO 액션".

> **장승웅(Big Blue)님**: 대표님은 1페이지에서 결정 가능해야 합니다
> "검토하겠습니다" 표현 절대 금지, "X 매출 예상, 비용 Y, GO 여부?" 형태로

**(속으로)**: 대표님 결정형 톤. 표 보여드리면 즉시 "한다 / 안 한다" 결정 나오게.

**분석 요청**:
```
@store_sales.csv @channel_sales.csv @campaign_calendar.csv
@templates/lush/exec-quarterly-briefing.md 관점으로
이 4주치 데이터를 "분기 누적"으로 가정해서 1페이지 임원 브리핑 만들어줘.
대표님이 1페이지에서 GO/NO-GO 결정 가능한 형태로.
특히 채널별 수수료 떼고 진짜 순매출이 얼마인지 강조해줘.
```

**결과 확인 후 추가 질문 예시**:
```
"다음 분기 액션 후보 2개를 각각 GO 시 예상 매출·비용·ROI로 정리해줘.
대표님이 표 보고 동그라미만 치면 결정 끝나는 형태로."
```

## 미션 2: 매장 인력 효율 부록 + 채널 자동화 제안

미션 1 마무리하고 17시 35분. 이재춘(Snow)·신은섭(Jason)이 요청한 두 가지를 부록에 압축으로 박아넣는다.

> **이재춘(Snow)님**: 매장 매출/인력 효율 한 줄이면 인사 재배치 논의 시작 가능합니다
>
> 매장 인력 수는 우리가 따로 가지고 있어요. 분기 매출만 매장별로 정리되면 제가 인력 효율 컬럼 채울게요

**(속으로)**: 인력 데이터까지 내가 만들 필요는 없다. 매장별 분기 매출 표 한 장만 깔끔하게 떨어트리면 Snow가 받아서 자기 데이터로 곱한다. 협업 지점이 명확하다.

**분석 요청**:
```
@store_sales.csv
매장 10곳의 분기 누적 매출 + 객수 + 객단가 표로 만들어줘.
이재춘(Snow, HRM)에게 전달할 거니까 매장 인력 효율 컬럼을 빈 칸으로 추가해줘.
신은섭(Jason, 하이테크)에게도 전달할 거니까 채널별 매출도 같은 형식으로 별도 표.
```

**결과 확인 후 추가 질문 예시**:
```
"신은섭(Jason)이 다음 분기부터 자동화하려고 하니까,
이 표를 만드는 데 필요한 데이터 소스·계산 로직을 한 단락으로 정리해줘.
데이터팀이 받아서 바로 시스템에 박을 수 있게."
```
